# 🥽 Low-End VR Guide

> Running Virtual Reality on hardware that shouldn't be able to run it.

---

## 🎯 Goal

This guide focuses on maximizing VR performance on low-end systems.

Supported hardware examples:

* Intel Integrated Graphics
* AMD APUs
* GTX 750 Ti
* GTX 950
* GTX 1050
* RX 460
* RX 560

---

## ⚠️ Expectations

VR on low-end hardware requires compromises.

You may need to:

* Reduce resolution
* Lower refresh rate
* Use lightweight games
* Apply aggressive optimizations

---

## 🔧 Performance Optimizations

### Windows

* Enable Game Mode
* Disable background apps
* Update GPU drivers
* Use High Performance power plan

### SteamVR

Recommended settings:

* Motion Smoothing OFF
* Resolution 50–80%
* Disable unnecessary overlays

## 🔥 Tips

1. Close every unnecessary application.
2. Use SSD storage whenever possible.
3. Keep GPU temperatures under control.
4. Lower render resolution before lowering graphics quality.
5. Test different OpenXR runtimes.

---

## Conclusion

VR does not require a high-end gaming PC to be enjoyable.

With proper optimization, even older hardware can provide immersive experiences.
